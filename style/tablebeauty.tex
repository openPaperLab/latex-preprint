\section*{Tablebeauty Usage Example}

Use \verb|\tbgroup| for full-width group header rows, \verb|\tbhighlight| for
important method rows, and \verb|\tbsubtle| for low-emphasis alternating rows.

\tbsetup{
  group-color=pink!15,
  highlight-color=blue!8,
  subtle-color=gray!8
}
